## Technologies
* Ruby
* Ruby on Rails
* React
* React Router
* Devise
* CarrierWave
* Fog
* MiniMagick
* Bootstrap

## Installation & Usage
```
git clone https://github.com/BrettRHolland/daytrippers.git
cd daytrippers
bundle install
npm install
rails db:create && rails db:migrate && rails db:seed
foreman start
```
